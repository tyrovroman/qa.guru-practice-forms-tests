# Automation Practice Form

Учебный проект автоматизированного теста для формы регистрации студента с использованием Java, Gradle, Selenide и JUnit 6.

## Описание

Этот проект реализует простой автотест для [практической формы](https://qa-guru.github.io/one-page-form/automation-practice-form.html).

**Сценарий теста:**
1.  Заполнение всех полей формы статичными данными.
2.  Отправка формы нажатием кнопки "Submit".
3.  Валидация данных в модальном окне путем сравнения их с введенными значениями.

---

## Технологии

| Технология | Назначение |
| :--- | :--- |
| **Java** | Язык программирования |
| **Gradle** | Система сборки и управления зависимостями |
| **Selenide** | Фреймворк для написания UI-тестов |
| **JUnit 6** | Фреймворк для запуска тестов |
